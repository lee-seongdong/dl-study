# NLP
자연어를 컴퓨터에게 이해시키기 위한 기술(벡터화)  
대부분의 기술이 `분포가설`에 기초한다.

> 분포가설: 단어의 의미는 주변 단어에 의해 형성된다

## 1. 통계 기반 기법
주변 단어(맥락)의 등장 횟수를 기반으로 한 기법  
표현되는 벡터는 희소벡터이기 때문에, 차원감소 기법이 필요  

> 차원 감소: 희소벡터를 밀집벡터로 표현하기 위한 기법. 대표적으로 `SVD`가 있음

### 특징
- 전체 데이터를 한꺼번에 처리하기 때문에, 학습 데이터가 커지면 자원을 많이 쓰게됨
- 데이터 추가 시 전체 데이터를 재학습 해야함
- 시간 복잡도 O(n^3)

### 1.1. 동시 발생 행렬
등장 횟수를 그대로 벡터화 한 행렬로 표현  
단순 등장 횟수만을 본다면, 의미를 제대로 포함할 수 없음('a', 'the', ...)

### 1.2. 양의 점별 상호정보량(PPMI)
단어간의 관련성을 등장 확률에 기반하여 표현  
PMI가 음의 무한대로 발산하는 것을 방지하기 위해 PPMI를 사용  
- P(x): x가 일어날 확률, P(x, y): x, y가 동시에 일어날 확률
- PMI(x, y) = log2(P(x, y) / P(x), P(y))
- PPMI(x, y) = max(0, PMI(x, y)) (두 단어의 동시발생 횟수가 0일때, PMI가 음의 무한대로 발산하는것을 방지)


## 2. 추론 기반 기법 (word2vec)
추론 문제를 학습하는 신경망을 이용한 기법  
학습된 은닉층의 가중치가 곧 단어의 분산 표현이 됨

> 은닉층의 입력가중치, 출력가중치 모두 분산표현으로 사용할 수 있지만, 주로 입력가중치만 사용함

### 특징
- 데이터 일부를 순차적으로 학습
- 데이터 추가 시 추가학습 가능
- 시간 복잡도 O(n)

### 2.1. CBOW (Continuous Bag of Words)
주변 단어들(맥락) → 중심 단어(target) 추론하는 모델  
입력층이 맥락 단어 수만큼 존재, 은닉층에서 평균을 취해 다음 계층으로 전달  
학습 완료 후 W_in의 각 행이 각 단어의 분산 표현  

### 2.2. Skip-gram
중심 단어(target) → 주변 단어들(맥락) 추론하는 모델 (CBOW의 반대 방향)  
일반적으로 CBOW보다 단어 분산 표현의 품질이 높음  
학습 데이터가 적거나 희귀 단어가 많을 때 유리  

### 모델의 개선 과정
- Embedding 레이어  
원핫벡터와 W_in의 곱은 W_in의 특정 행을 꺼내는것과 동일  
`W_in[word_id]` 인덱싱으로 계산비용 제거
- 네거티브 샘플링  
다중 분류 문제를 이진 분류 문제로 변환
    - 기존: 전체 V개 중 한 단어를 추론하기 위해 전체 단어에 대해 `softmax`를 계산. 계산량은 O(V)
    - 개선: 정답단어 + 무작위 오답 k개 단어에 대해 `sigmoid`를 계산. 계산량은 O(k). k는 보통 5~20  
    오답 단어는 완전 무작위가 아닌 **등장 빈도에 비례**하여 샘플링

## 3. 전이 학습 (Transfer Learning)
단어벡터(학습된 분산 표현)를 다른 자연어 처리 작업에 활용하는 기법  
일반적으로 대규모 코퍼스로 사전 학습된 단어벡터를 활용함  
- 방식 1: 임베딩 가중치를 **고정(freeze)** 하고 이후 계층만 학습
- 방식 2: 임베딩 가중치도 **fine-tuning** (태스크에 맞게 미세 조정)
